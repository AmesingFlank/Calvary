# Cavalry
A GPU Ray Tracer


> Cheers love, the cavalry's here!  -- Tracer (Lena Oxton)

Features:

* GPU Bounding Volume Hierarchy construction using radix trees (Aila 2012)

* (Partial) compatibility with PBRT scene definition files.

* Halton sampler
